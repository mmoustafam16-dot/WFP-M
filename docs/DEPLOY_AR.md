# خطوات النشر الاحترافية

## 1) GitHub

ارفع المشروع على GitHub repository خاص.

## 2) Vercel

- افتح Vercel.
- Import Project من GitHub.
- اختر Next.js.
- أضف Environment Variables من `.env.example`.
- Deploy.

## 3) Supabase

- أنشئ مشروع Supabase.
- شغّل `supabase/schema.sql`.
- من Project Settings > API انسخ:
  - Project URL
  - anon public key
- ضع القيم في Vercel Environment Variables.

## 4) التحقق

جرّب:

- إنشاء حساب.
- تسجيل الدخول.
- إنشاء مشروع جديد.
- حفظ المشروع.
- ظهوره في Dashboard.
- إرسال رسالة من Contact.
