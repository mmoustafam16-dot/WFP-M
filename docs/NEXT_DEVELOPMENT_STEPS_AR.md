# التطوير التالي المقترح

## 1) ربط الحفظ مباشرة بنتائج البرنامج

النسخة الحالية تحفظ المشروعات من صفحة Save Project. المرحلة التالية هي ربط صفحة البرنامج الحالية بزر مباشر يرسل المدخلات والنتائج إلى قاعدة البيانات.

## 2) تقارير PDF احترافية

إضافة API endpoint لتوليد تقرير PDF يحتوي على:

- بيانات المشروع.
- المدخلات.
- نتائج Green/Blue/Grey WF.
- الرسوم والجداول.
- المراجع العلمية.

## 3) لوحة إدارة Admin

إضافة صلاحيات لإدارة:

- بيانات المحاصيل.
- الدول.
- المحافظات.
- المعاملات العلمية.
- المستخدمين.

## 4) اشتراكات مدفوعة لاحقًا

إضافة خطط:

- Free
- Researcher
- Consultant
- Enterprise
