# Water Footprint Decision Tool - Full Stack Website

Professional full-stack edition with:

- Landing page
- Login/register
- User dashboard
- Online project saving with Supabase PostgreSQL
- Embedded current calculator
- Contact email form via Resend
- Domain and deployment documentation

## Local run

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000

## Database

Create a Supabase project, open SQL Editor, and run `supabase/schema.sql`.

## Deployment

Deploy to Vercel and connect a domain/subdomain such as `wfp.yourdomain.com`.
